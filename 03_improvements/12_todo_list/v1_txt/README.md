# To-Do List V1 — TXT Storage

A simple command-line to-do list app
built with Python and txt storage.

## What It Does
- Add new tasks to your list
- View all current tasks in a numbered list
- Mark a task as complete and remove it
- Clear all tasks at once
- All tasks saved permanently to a txt file

## What I Learned
- Building a complete task management flow —
  add, view, complete and clear
- Showing completion indicator `[x]` before
  removing a task — thinking about user feedback
- Using the read, modify, write back pattern
- Accessing a specific item with `tasks[index]`
  before removing to show what was completed
- Using `pass` to clear a file cleanly

## How To Run
```bash
python todo_list_v1.py
```
