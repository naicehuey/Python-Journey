# Contact Book V2 — JSON Storage

A fully featured command-line phone book app
built with Python and JSON storage.

## What It Does
- Add new contacts with name and phone number
- View all contacts, favorites only, or sorted
- Search contacts by name or phone number
- Mark and unmark contacts as favorites
- Edit existing contacts
- Delete contacts
- Show contact statistics (total, favorites, others)
- All data saved permanently using JSON

## What I Learned
- Using JSON for structured data storage
- Storing dictionaries with multiple fields
- Using `lambda` functions for custom sorting
- Using `sorted()` with a key to sort alphabetically
- Using `.get()` for safe dictionary access
- Filtering and sorting using function parameters
- Duplicate detection before adding new contacts
- Tracking stats by looping through data
- Using emoji in output for better user experience

## How To Run
```bash
python contact_book_v2.py
```
