# Repeat Memory — Improvements

A simple command-line book manager built
with Python and JSON storage.
Built to practice and reinforce the reusable
function pattern.

## What It Does
- Add books with name, author and release year
- View all books in formatted list
- Search books by name
- Delete a specific book
- All data saved permanently using JSON

## What I Learned
- Reinforcing the reusable pattern —
  `save_data()`, `load_data()`, `display_list()`
  written from memory without reference
- That repeating patterns deliberately builds
  muscle memory — you stop thinking about syntax
  and start thinking about logic
- Always return after invalid input validation
  to prevent code continuing after an error
- Using unique filenames per project to prevent
  data mixing between programs

## Note
This project was built deliberately to repeat
and memorize the reusable function pattern —
repetition is how patterns become instinct.

## How To Run
```bash
python repeat_memory.py
```
