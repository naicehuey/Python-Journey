# Most Common File Type — OS Practice

An upgraded file counter that finds which file type
appears most in a folder. Built by adding original
thinking on top of the counting practice.

## What It Does
- Scans a folder and counts files by type
- Uses `max()` to find the highest count
- Reports which file type dominates the folder
- Shows the total count of the most common type

## What I Learned
- Using Python's built-in `max()` function
- Passing multiple variables into `max()` to find
  the highest value among them
- Comparing a variable against `max()` result
  to identify which category won
- Adding creative features beyond what was being taught
- Thinking about what useful information a program
  can extract from data

## Known Limitation
If two file types have equal counts only the first
one is reported — ties are not detected.
Could be improved using a dictionary approach.

## Why This One Is Special
Nobody told me to add the `max()` feature —
this was original thinking on top of practice.
That is the moment a beginner starts thinking
like a developer.
