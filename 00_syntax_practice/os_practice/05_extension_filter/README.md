# Extension Filter — OS Practice

Testing how to filter and display only specific
file types from a folder.

## What It Does
- Scans a folder for files
- Filters and displays only image files (.png, .jpg)
- Numbers each result using enumerate()

## What I Learned
- Filtering files by extension using a list `[".png", ".jpg"]`
- Checking if a value exists in a list using `in`
- Combining `enumerate()` and `os.path.splitext()` together
- Using `\n` inside f-strings for formatted output
- Checking extension from full path vs just filename

## What This Led To
This was the final practice piece before building
the full File Organizer project — every `os` concept
practiced before this came together here.
