# Basic Calculator — Beginner

A simple command-line calculator built with Python.

## What It Does
- Takes two numbers as input from the user
- Takes a mathematical operator (+, -, *, /)
- Performs the calculation and displays the result
- Handles unsupported operators gracefully

## What I Learned
- Taking multiple inputs from the user
- Converting input to integer using `int()`
- Using `if/elif/else` conditional statements
- Performing basic arithmetic operations in Python
- Thinking about edge cases — what happens when
  the user enters an unsupported operator
- That programs should handle unexpected input
  gracefully instead of crashing

## How To Run
```bash
python calculator.py
```
