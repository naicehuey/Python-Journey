# Send Your Money Safe — Beginner

A simple command-line money transfer app built with Python.

## What It Does
- Welcomes the user to the platform
- Displays the current date and time
- Validates that the user enters a valid number
- Keeps asking until a valid input is given

## What I Learned
- Importing and using Python modules (`datetime`)
- Using `while` loops to repeat actions
- Using `try/except` for error handling and input validation
- Using a boolean flag to control a loop
- Using `float()` for decimal number conversion
- That programs should never crash on bad input —
  always validate and ask again

## Known Limitations
- Card list exists but transfer functionality
  not yet implemented
- Project was the beginning of a bigger idea

## How To Run
```bash
python send_money.py
```
