# Hello World — Day One

The very first Python program. Every developer starts here.

## What It Does
- Prints a welcome message to the screen
- Stores personal information in variables
- Combines variables and strings into a sentence

## What I Learned
- Writing and running my first Python program
- Creating and storing data in variables
- String concatenation using `+` to join text together
- The difference between hardcoded text and variables
- That every programming journey starts with
  just one line — `print("Hello World")`

## Note
This is where everything began. No functions, no loops,
no logic — just the excitement of making something
appear on a screen for the very first time.

## How To Run
```bash
python hello_world.py
```

## Output
```
Hello World
My name is Glethu, Me and my brother Atupele,
we both wanna be Programmers
```
