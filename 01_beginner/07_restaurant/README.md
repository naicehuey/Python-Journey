# KiMeFra Restaurant — Beginner

A command-line restaurant ordering system built with Python.
Named after people close to the developer.

## What It Does
- Displays a menu with prices
- Takes a meal order from the user
- Calculates tip between 10 and 20 percent
- Adds the tip to the meal price
- Offers optional toppings (Tomato or Cheese spray)
- Calculates and displays the final total

## What I Learned
- Using list indexing to access specific items (`list1[0]`)
- Performing percentage calculations in Python
- Using `str()` to convert numbers for string concatenation
- Building multi-step programs with chained decisions
- Modeling a real world business concept in code
- That programs can represent real life situations

## Known Limitations
- No functions yet — everything written linearly
- Program crashes if invalid meal chosen then topping selected
- Tip percentage not validated (accepts values outside 10-20)
- Only two menu items available

## How To Run
```bash
python restaurant.py
```
