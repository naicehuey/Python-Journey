# Patient Register — Beginner

A simple command-line patient registration program
built with Python.

## What It Does
- Takes a new patient's name as input
- Takes the patient's age as input
- Displays the patient's information in a formatted output

## What I Learned
- Taking user input using `input()`
- Converting input to integer using `int()`
- Using f-strings `f"..."` for cleaner string formatting
- Using `\n` inside f-strings to display output
  on separate lines
- Taking different types of input — string and integer
- Moving away from hardcoded values to dynamic user input

## Growth From Previous Project
Unlike Hello World where everything was hardcoded,
this project takes real input from the user —
making the program dynamic for the first time.
The jump from `+` concatenation to f-strings
also shows immediate style improvement.

## How To Run
```bash
python patient_register.py
```
