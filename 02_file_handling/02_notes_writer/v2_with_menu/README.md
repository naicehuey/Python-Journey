# Notes Writer V2 — With Menu

An upgraded command-line notes app built with Python.

## What It Does
- Option 1: Start a brand new note (replaces existing)
- Option 2: Continue adding to existing notes
- Option 3: View all saved notes with numbered lines
- Displays updated notes automatically after adding

## What I Learned
- Using `enumerate()` to number items while looping
- Using `for` loops to read and display file content
- Using `.strip()` to clean up whitespace and newlines
- Building a menu-driven program with file handling
- Using `exit()` to control program flow
- That giving users control through a menu makes
  programs feel complete and professional

## Growth From V1
V1 was a basic linear writer with no user control.
V2 gives the user full control through a menu —
start fresh, continue writing, or view notes anytime.

## How To Run
```bash
python notes_writer_v2.py
```
